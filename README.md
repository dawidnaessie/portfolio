# Portfolio - Next.js

Mój osobisty projekt portfolio stworzony w Next.js, pokazujący moje projekty, umiejętności i doświadczenie.

**Live demo:** [https://portfolio-pink-nu-79.vercel.app/](https://portfolio-pink-nu-79.vercel.app/)

---

## Technologie
- **Next.js** (React)  
- **TypeScript / JavaScript**  
- **Tailwind CSS / CSS Modules / Styled Components**  
- **Vercel** (hosting)
- **Vercel v0** (scripting)

---

## Instalacja i uruchomienie lokalne

Sklonuj repozytorium:
```bash
git clone https://github.com/dawidnaessie/portfolio.git
cd portfolio
```
Zainstaluj zależności:
```bash
npm install
# lub
yarn
```
Uruchom projekt w trybie deweloperskim:
```bash
npm run dev
# lub
yarn dev
```
Aby zbudować projekt do produkcji:
```bash
npm run build
npm run start
# lub
yarn build
yarn start
```
Projekt będzie dostępny pod adresem: http://localhost:3000
